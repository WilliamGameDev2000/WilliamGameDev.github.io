What I bring to the table

  • I have strong programming skills and knowledge covering core programming principles and specific techniques for C# and C++ languages. I am also capable of programming using game design patterns such as OOP, entity component systems and working in methods such as agile programming.
  • I have used and can program in game engines such as Unity and Unreal, and C++ frameworks such as DirectX and OpenGl based engines. This gives me flexebility in creating and approaching game development and what engine I can use.
  • I have an understanding of 3d modelling and game art which allows me to work with both describing the aesthetic and keeping it within scope of a project technically.
  • My experience with playing and designing games has given me a good ability to think through a game and a level and how those features can all be tied together. I understand theory such as the flow state theory and how that informs level design to be satisfying.
  • I am constantly learning and innovating upon my skillset, in order to make the best games possible.
  • I have a degree from UWE's TIGA accredited Games Technology course and I have a skills bootcamp programming certificate from Mastered.
